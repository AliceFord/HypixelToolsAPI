# nameWrapper

## Description

Format name of hypixel player with their rank.

## Parameters
- name

## Example GET Request
`GET / hypixeleaderboards.com/api/nameWrapper?name=dev`

## Example Response

```
{
  "html":"<span style=\"font-family: 'Minecraftia';\"><span style=\"text-shadow: 1px 1px #eee; color:#AAAAAA\"> dev</span></span>"
}
```
